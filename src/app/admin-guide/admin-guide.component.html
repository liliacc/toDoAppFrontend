Kā uzstādīt web lapu Amazon serverī

  1. Vispirms jādodas uz Amazon servera mājaslapu
  2. Jāreģistrējas – klikšķina uz Sign Up
  3. Izvēlas free Amazon Web Services account
  4. Services izvēlnē klikšķina uz S3
  5. Jaunajā logā izvēlas Create bucket
  6. Norāda bucket vārdu un reģionu
  7. Tālāk atzīmē Permissions jeb atļaujas ja vajadzīgs
  8. Klikšķina uz Create Bucket
  9. Pēc novadīšanas uz S3 lapu, izvēlas Bucket name, ko tikko uztaisīja
  10. Iet uz sadaļu propeties jeb īpašības.
  11. Izvēlas Static website hosting
  12. Aizpilda laukus ievadot faila nosaukumu, kas atbilst index.html
